Description

This digitisation project is the result of the work carried out by Esther Pocock and Francisco Ferrer, students of a Masters program in Digital Library at Högskola Borås. The original book was loaned by the library of Chalmers Univertity in Gothenburg. The main aim of this project was to preserve the content of the original source, as it is the only known copy in existence. more chapters to be added.
